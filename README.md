# MailNavigator — releases

Официальные установщики надстройки **MailNavigator** для Microsoft Outlook (Classic).

- Скачать последнюю версию: **[Releases](https://github.com/Danieli4/MailNavigator-releases/releases/latest)** → `MailNavigatorSetup-<версия>.exe`
- Установка: закройте Outlook, запустите `MailNavigatorSetup-<версия>.exe`.
- В каждом релизе указана контрольная сумма SHA256 установщика.

В этом репозитории нет исходного кода — только исполняемые файлы релизов.
Публикация выполняется автоматически CI-пайплайном основного (приватного) репозитория.

---

Official installers for the **MailNavigator** Outlook add-in. Binaries only, no source code.
